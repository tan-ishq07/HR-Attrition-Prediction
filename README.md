#  HR Analytics and Employee Attrition Prediction

##  Project Overview

This project focuses on **Data Analytics and Predictive Modeling** to understand and analyze employee attrition patterns within an organization.

The primary goal is to:

* Perform **Exploratory Data Analysis (EDA)** on HR data
* Identify key factors influencing employee attrition
* Build a **predictive model** based on insights derived from the analysis
* Develop a **GUI-based system** to make real-time predictions

---

## Dataset

The dataset used in this project contains various employee-related attributes such as:

* Age, Department, Job Role
* Monthly Income, Daily Rate
* Work Experience, Training, Performance
* Job Satisfaction, Work-Life Balance, Environment Satisfaction

These features are analyzed to determine their impact on employee attrition.

---

##  Data Analysis (EDA)

The project begins with **in-depth data analysis**, including:

* Distribution analysis of key variables
* Correlation between features and attrition
* Identification of high-risk employee segments
* Visualization using graphs and charts

Key insights derived from analysis:

* Employees with lower job satisfaction and work-life balance show higher attrition
* Overtime and lower income levels significantly impact attrition probability
* Early-career employees are more likely to leave

---

##  Model Development

Based on the insights obtained from data analysis, a **Logistic Regression model** is developed to predict employee attrition.

### Steps involved:

1. Data Cleaning and Preprocessing
2. Encoding categorical variables
3. Train-test split of dataset
4. Model training using Logistic Regression
5. Model evaluation using accuracy score

The model is designed to reflect patterns discovered during the analytical phase, making it both **data-driven and interpretable**.

---

##  GUI Implementation

A **Tkinter-based GUI** is developed to enhance usability.

### Features:

* User-friendly input form for employee details
* Real-time prediction of attrition
* Display of:

  * Prediction result (Stay / Leave)
  * Risk Level (Low / High)
  * Model Accuracy
* Visualization of prediction probability using graphs

This transforms the analytical model into a **practical decision-support tool**.

---

##  Key Highlights

* Combines **Data Analytics + Machine Learning**
* Focus on **insight-driven modeling** rather than only prediction
* Interactive GUI for real-world usability
* Clean and structured workflow from data → insights → model → application

---

##  Conclusion

This project demonstrates how **data analytics can be used to derive meaningful insights** and build predictive solutions.

By analyzing employee data and developing a model based on those insights, organizations can:

* Identify high-risk employees
* Take preventive actions
* Improve employee retention strategies
